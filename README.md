# 3D Cube in Python

Simple rotating 3D cube made with Python `turtle`.

* No OpenGL
* No Shaders
* Uses basic math projection
* Pure Python

## Requirements

```bash
pip install turtle
```

## Run

```bash
python cube.py
```

## Features

* Rotating 3D cube
* Perspective projection
* Wireframe rendering
* Lightweight and simple
